今回のリニューアルで、「新しい車ごとにHTMLページを作る手間」をなくし、「データ（data.js）を書き換えるだけでサイト全体が自動更新される」という、非常にメンテナンスしやすいモダンな仕組みを採用しています。

今後のサイト運用・メンテナンスの手順をわかりやすくまとめました。これをマニュアルとして保存しておくと便利です！

🐾 1. 普段の更新（車の追加・編集・売約済み処理）

日常的な更新は、すべて data.js のファイルを編集するだけで完結します。HTMLファイルを開く必要はありません。

① 新しい在庫車を追加する

data.js の carData =[ の中に、以下のブロックをコピー＆ペーストして内容を書き換えます。

    {
        id: "固有の英数字（例: wagonr_01）", // URLに使われるので他の車と被らないように！
        name: "車種名",
        catchphrase: "一覧や詳細のトップに出るキャッチコピー",
        price: "価格", // 数字だけ（例: 45.8）
        status: "販売中", // 「販売中」「商談中」「売約済」のいずれか
        mainImage: "画像のURL", // 一覧に表示されるメイン画像
        
        specs:[ // スペック表（項目は自由に増やせます）
            { title: "年式", text: "2020年モデル" },
            { title: "走行距離", text: "5.5万km" }
        ],

        freeText: `
            <h3>自由記述の見出し</h3>
            <p>ここにアピールポイントなどを書きます。</p>
            <figure class="c mb30">
                <img src="追加の画像URL" alt="写真">
            </figure>
        `
    }, // ← ★最後は必ずカンマ( , )で終わるのを忘れずに！

② 車が売れた！「販売実績」へ移動する

車が売れたら、わざわざページを作り直したり移動させたりする必要はありません。 data.js を開き、その車の status を書き換えるだけです。

  - 変更前: status: "販売中", または status: "商談中",
  - 変更後: status: "売約済",

これだけで、自動的に「在庫車一覧（product.html）」から姿を消し、「販売実績（sold.html）」のページに移動します。商談中の場合は
status: "商談中", とすれば、一覧画面でピンクの「商談中」バッジが付きます。

🐾 2. 写真（画像）の準備について

data.js の mainImage や、freeText の中に入れる写真は「画像のURL（https://～.jpg）または相対パス」で指定します。



🐾 3. たまにやる更新（ご挨拶や会社概要の変更）

お店の基本情報や、トップページの文章を変えたい場合は、該当するHTMLファイルを直接編集します。

  - トップページの「ごあいさつ」や「選ばれる理由」を変えたい: 👉 index.html を開き、<div class="content-wrapper">
    の中にあるテキストを書き換えます。
  - 「会社概要」や「スタッフ紹介（にゃんこ部長など）」を変えたい: 👉 about.html を開き、テキストを書き換えます。

⚠️ メンテナンス時の重要な注意点（よくあるエラー）

プログラミングの記述ルールによるエラーを防ぐためのポイントです。もし「画面が真っ白になった」「車が表示されなくなった」という場合は、以下をチェックしてください。

1.  カンマ , の忘れ／余分なカンマ 車と車のデータの間や、specs の項目の最後には必ず半角カンマ ,
    が必要です。逆に、一番最後の項目の後ろに余分なカンマがあるとエラーになることがあります。
2.  クォーテーション " と バッククォート `
      - 短い文字（名前や価格など）はダブルクォーテーション " で囲みます。
      - 長い文章（freeText）は、改行ができるバッククォート ` （Shift ＋ @キー
        などで入力）で必ず囲んでください。これを消してしまうとサイトが真っ白になります。
3.  更新したのに画面が変わらない場合 ブラウザが古いデータを記憶している（キャッシュ）ことが原因です。パソコンの場合は
    Ctrlキー（MacはCommand）を押しながら F5キー（または更新ボタン） を押して「スーパーリロード」を行ってください。

この仕組みにより、ご自身で「車を仕入れたら data.js に追加してアップロードするだけ」という非常にスピーディな運用が可能です。
